# IBM Data Analyst Capstone Project

This repository contains my capstone project completed as part of the **IBM Data Analyst Professional Certificate**.

## Project Overview

The goal of this project is to analyze technology trends using survey and job market data.
The project includes data collection, data cleaning, exploratory data analysis, and visualization.

## Project Workflow

1. Data collection using APIs
2. Web scraping
3. Data exploration
4. Data cleaning and transformation
5. Exploratory data analysis
6. Data visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Repository Structure

```
notebooks/  – analysis notebooks  
data/       – datasets used in the analysis  
reports/    – final capstone project report  
```

## Result

The project demonstrates practical skills in collecting, preparing, analyzing, and visualizing real-world data.
